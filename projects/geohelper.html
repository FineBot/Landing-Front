GeoHelper

Описание

Проект для реализации AR-слоя мира, в котором отображаются объекты дополненной реальности в зависимости от заданного географического положения. Проект состоит из сайта и приложений для платформ Android и IOS. Сайт предназначен для менеджмента данными, представляемых в AR-слое. Мобильное приложение служит для отображения AR-слоя мира на экране используемого устройства.
